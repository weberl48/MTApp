# PWA Icons

Add your app icons here:

- `icon-192.png` - 192x192 pixels (required for PWA)
- `icon-512.png` - 512x512 pixels (required for PWA)

These icons should be:
- Square format
- PNG format
- Include padding for "maskable" display (safe zone is inner 80%)

You can generate these from a logo using tools like:
- https://realfavicongenerator.net/
- https://maskable.app/editor
